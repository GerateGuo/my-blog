# 摄影作品集

这里展示我的摄影作品。每一张照片都记录了一个独特的瞬间。

## 使用方法

1. 将照片放入 `source/images/photography/` 目录
2. 在下方添加照片信息，使用以下格式：

```markdown
{% gallery %}
  ![照片标题](/images/photography/your-photo.jpg)
{% endgallery %}
```

或者使用 Markdown 格式：

```markdown
![照片标题](/images/photography/your-photo.jpg)
*照片标题 - 拍摄地点*
```
